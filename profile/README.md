## Sobre Nos

A **LP Solutions Group** e uma empresa de tecnologia focada em criar ferramentas inteligentes para vendedores e analistas de e-commerce. Combinamos automacao, analise de dados e interfaces intuitivas para transformar dados brutos em decisoes estrategicas.

## Nossos Projetos

### Shopee Spy

Extensao Chrome (MV3) que injeta dados de inteligencia competitiva diretamente nas paginas da Shopee Brasil.

**Funcionalidades principais:**

- Estimativa de vendas e faturamento por produto
- Ranking de vendedores e categorias
- Calculadora de lucro integrada
- Dashboard de analise competitiva

**Tech Stack:** JavaScript · CSS · HTML · Chrome Extensions API (Manifest V3)

## Como Trabalhamos

| Pratica | Descricao |
|---------|-----------|
| **Git Flow** | Branches pessoais → `dev` → `main` com PRs revisados |
| **CI/CD** | Linting automatizado via GitHub Actions em cada PR |
| **Projeto** | Kanban Board no GitHub Projects para rastrear tarefas |
| **Issues** | Bugs, features e melhorias documentados com labels e milestones |
| **Code Review** | Todo codigo passa por revisao antes de entrar em `dev` |
